How to Use the Trade Calculator
  • Enter the amount of each ore you want to offer in the left section.
  • The calculator will show you the equivalent value of each ore you can receive on the right.
  • All calculations are automatic as you type.
  • Use the switch below to toggle between exact and rounded values.
  • Adjust multipliers for each ore by clicking in the ore icons

Your Offer - Help
  • Enter the amount of each ore you want to include in your offer.
  • Click on any ore icon to set a custom multiplier for that specific ore.
  • The multiplier affects how the ore value is calculated in the equivalent exchange.
  • You can mix different ore types to create your desired trade offer.
Equivalent Exchange - Help
  • This section shows the equivalent amounts you can receive for each ore type based on your offer.
  • Values are automatically calculated as you type in the "Your Offer" section.
  • Blue values show exact calculations.
  • Green values show rounded amounts (toggle with the switch at the top).
  • These values help you determine fair trades for different ore types.
